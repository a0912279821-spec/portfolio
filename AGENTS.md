# AGENTS.md

## 项目说明

这是 Jerry 的个人项目展示主页（portfolio），由 GitHub Pages 托管运行。
在线地址：https://a0912279821-spec.github.io/portfolio/

## 技术栈

项目使用原生 Web 技术构建，不依赖任何框架或构建工具：

- HTML
- CSS
- JavaScript

## 修改规则

### 必须遵守

1. **不要引入框架**：禁止引入 React、Vue、Vite、Tailwind、Bootstrap 等任何第三方框架或构建工具。保持原生技术栈。
2. **不要直接改 main**：所有修改必须新建分支，通过 Pull Request 合并。
3. **保持简洁风格**：页面风格保持简洁、清晰，适合作为大一学生的 GitHub 项目主页。不要过度设计。
4. **保持现有结构**：修改页面时优先在现有 HTML/CSS/JS 结构上增量调整，不要大幅重构。
5. **不要删除已有内容**：除非明确要求，否则不删除页面已有内容。
6. **不要添加复杂效果**：不要添加复杂动画、复杂 JavaScript 逻辑、或任何增加页面负担的特性。

### 修改后检查清单

- [ ] 页面能正常在浏览器打开
- [ ] 所有链接正确可用
- [ ] 手机端（≤680px）布局正常
- [ ] 电脑端布局正常
- [ ] 没有引入新的依赖或框架

## 工作流程

1. 从 `main` 创建新分支
2. 进行修改
3. 提交并推送
4. 使用 `gh pr create` 自动创建 Pull Request
