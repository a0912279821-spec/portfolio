## 如何打开

将仓库下载到本地后，直接用浏览器打开 `index.html` 即可查看页面。

也可以在项目目录运行一个简单的本地服务器：

```sh
python3 -m http.server 8000
```

然后在浏览器访问 `http://localhost:8000`。
